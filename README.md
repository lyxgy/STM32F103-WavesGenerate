# 使用说明

每次切换都会重置设置的波形数据(频率、幅度)，方波的占空比除外

## 串口指令

切换波形：WAVE:要切换的波形名字

调整幅度：AMP:数字(当前幅度的百分之多少)

调占空比：DUTY:数字(1-100)
